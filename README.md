🧩 Known Issues & Fixes
🖼️ Images not loading
Ensure correct relative paths
Confirm images are pushed to GitHub repo

📄 Resume download not working
Check file name consistency (resume.pdf)
Verify correct anchor href path
🔄 Changes not reflecting
Hard refresh browser (Ctrl + Shift + R)
Rebuild deployment on hosting platform
